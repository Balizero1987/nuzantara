How it works

Click either segment to switch themes. The active segment is highlighted with a gradient background and shows which mode is currently active.

Day Mode Active: Day segment has red-to-orange gradient background

Night Mode Active: Night segment has purple-to-cyan gradient background with neon glow

Pros: Shows both options simultaneously, very clear state indication, familiar iOS-style pattern

Cons: Takes up more header space (~100px), may feel cluttered on mobile
