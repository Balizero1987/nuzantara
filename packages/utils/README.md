# @nuzantara/utils

Shared utility functions for the Nuzantara monorepo.

## Usage

```typescript
import { capitalize, slugify, unique, chunk } from '@nuzantara/utils';
```

## Available Utilities

- **String**: `capitalize`, `slugify`
- **Array**: `unique`, `chunk`
- **Object**: `omit`
- **Date**: `formatDate`
