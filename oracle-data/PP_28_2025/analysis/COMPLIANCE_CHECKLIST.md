# PP 28/2025 Compliance Checklist

## 1. Registration & NIB

**Mandatory**: True

### Steps:

1. Register in OSS system
2. Obtain NIB
3. Input KBLI 5-digit code

**References**: Pasal 4, Pasal 211

---

## 2. Risk-Based Licensing

**Mandatory**: True

### Steps:

1. Determine business risk level
2. Low risk: Sertifikat Standar only
3. Medium risk: Sertifikat Standar + compliance
4. High risk: Izin required

**References**: Pasal 1, Pasal 2

---

## 3. Environmental Compliance

**Mandatory**: Depends on activity

### Steps:

1. Check if AMDAL required
2. If not AMDAL: prepare UKL-UPL
3. If minimal: SPPL self-declaration
4. Obtain Persetujuan Lingkungan (PL)

**References**: Pasal 15, Pasal 16

---

## 4. Location & Building

**Mandatory**: If physical location

### Steps:

1. Obtain KKPR (location suitability)
2. If construction: obtain PBG
3. After construction: obtain SLF

**References**: Pasal 27, Pasal 28

---

## 5. Foreign Workers (TKA)

**Mandatory**: If hiring foreigners

### Steps:

1. Submit RPTKA
2. System forwards to OSS and Imigrasi
3. Obtain work permit approval

**References**: Pasal 212

---

## 6. Ongoing Compliance

**Mandatory**: True

### Steps:

1. Maintain licenses validity
2. Submit periodic reports
3. Allow government inspections
4. Update data if changes

**References**: BAB VII - Pengawasan

---

## 7. Penalties

**Mandatory**: Yes

### Penalties:

- **Operating without license**: Operations stopped
- **False information**: License revocation
- **No SLF**: Building closure

**References**: BAB XI - Sanksi

---
