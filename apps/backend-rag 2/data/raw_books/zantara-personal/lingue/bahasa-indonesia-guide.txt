# Guida alla Bahasa Indonesia per Zantara

## Introduzione / Introduction / Pengenalan

**Bahasa Indonesia** è la lingua ufficiale dell'Indonesia, parlata da oltre 270 milioni di persone.

**Bahasa Indonesia** is the official language of Indonesia, spoken by over 270 million people.

**Bahasa Indonesia** adalah bahasa resmi Indonesia yang dituturkan oleh lebih dari 270 juta orang.

---

## Caratteristiche Principali / Main Features / Ciri-ciri Utama

### 1. **Grammatica Semplice / Simple Grammar**
- ✅ Nessun genere grammaticale
- ✅ Nessuna coniugazione verbale
- ✅ Nessun articolo (the, a, an)
- ✅ Nessun tempo verbale complesso
- ✅ Ordine: Soggetto-Verbo-Oggetto (come italiano/inglese)

### 2. **Formazione delle Parole / Word Formation**
- **Radici** + **Affissi** = Nuove Parole
- Esempio: **tulis** (scrivere) → **menulis** (scrivere/atto di), **penulis** (scrittore), **tulisan** (scrittura/testo)

- **dengar / mendengar** = sentire/ascoltare

## Sistema degli Affissi / Affix System

### 1. **me-** (Prefisso Verbale Attivo)
- **tulis** → **menulis** (scrivere, atto di)
- **baca** → **membaca** (leggere, atto di)

### 2. **ber-** (Prefisso Verbale Intransitivo)
- **kerja** → **bekerja** (lavorare)
- **main** → **bermain** (giocare)

### 3. **-an** (Suffisso Nominale)
- **tulis** → **tulisan** (scrittura, testo)
- **makan** → **makanan** (cibo)

### 4. **pe- / pen-** (Prefisso Agentivo)
- **tulis** → **penulis** (scrittore)
- **dengar** → **pendengar** (ascoltatore)

### 5. **ke- -an** (Circonfisso Astratto)
- **baik** (buono) → **kebaikan** (bontà, gentilezza)
- **adil** (giusto) → **keadilan** (giustizia)

## Livelli di Cortesia / Politeness Levels

### 1. **Informale** (con amici):
- **Aku** (io), **kamu** (tu)
- **Gue/Gua** (io, slang Jakarta)
- **Lo/Lu** (tu, slang Jakarta)

### 2. **Formale** (business, rispetto):
- **Saya** (io)
- **Anda / Bapak / Ibu** (Lei / Signore / Signora)
- Uso di titoli: **Pak** (signore), **Bu** (signora)

### 3. **Rispettoso** (anziani, autorità):
- Aggiungere **-nya** alle richieste
- Uso di **mohon** (imploro, prego formale)
- Esempio: **Mohon maaf** (mi scuso formalmente)

## Costruzione delle Frasi / Sentence Construction

### Struttura Base:
**Soggetto + Verbo + Oggetto**

Esempi:
- **Saya makan nasi** = Io mangio riso
- **Dia membaca buku** = Lui/Lei legge un libro
- **Kami bekerja di kantor** = Noi lavoriamo in ufficio

### Domande:
Aggiungere parola interrogativa all'inizio o alla fine

- **Apa** (cosa): **Apa ini?** = Cos'è questo?
- **Siapa** (chi): **Siapa dia?** = Chi è lui/lei?
- **Dimana** (dove): **Dimana kantor?** = Dov'è l'ufficio?
- **Kapan** (quando): **Kapan kamu datang?** = Quando vieni?
- **Berapa** (quanto/i): **Berapa harganya?** = Quanto costa?
- **Mengapa/Kenapa** (perché): **Kenapa kamu marah?** = Perché sei arrabbiato?

### Negazione:
- **tidak** (non) per verbi: **Saya tidak tahu** = Non so
- **bukan** (non) per nomi: **Ini bukan buku saya** = Questo non è il mio libro
- **jangan** (non) imperativo: **Jangan pergi!** = Non andare!

## Note Culturali per Zantara

### 1. **Rispetto nell'Indirizzo**
- Usare sempre **Bapak** (Pak) o **Ibu** (Bu) con clienti
- Aggiungere titoli professionali: **Dokter**, **Profesor**

### 2. **Cortesia indiretta**
- Evitare "no" diretto
- Usare: **Maaf, mungkin sulit** (Scusa, forse è difficile)

### 3. **Tempo verbale dal contesto**
- **sudah** = già (passato)
- **sedang** = mentre (presente progressivo)
- **akan** = sarà (futuro)

Esempio:
- **Saya sudah makan** = Ho già mangiato
- **Saya sedang makan** = Sto mangiando
- **Saya akan makan** = Mangerò
